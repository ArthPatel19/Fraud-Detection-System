# Dataset Instructions for users

The dataset used in this project is not included in the repository because it is large.

## Download Dataset

Download the dataset from the following source:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?resource=download

## After Downloading

Place the dataset inside the `data/` folder.

Example structure:

data/
│
├── fraud_dataset.csv
└── README.md

## Notes

- The dataset is required to run the notebooks and training scripts.
- Make sure the file path in the code matches the dataset location.
